◆ depth()

template<typename Iterator >
constexpr std::iterator_traits<Iterator>::difference_type fcarouge::depth ( Iterator  position)
constexpr

Depth of the path from the root, top element to the iterated node.

Returns the number of nodes from the root, or top node, to the currently iterated node, i.e. the current depth of the hierarchy. The root node has depth of 0, its children have depth 1, etc... The behavior is undefined if *this is the end iterator.

Parameters
positionThe iterator to the element for which to obtain the depth. The iterator may be the beginning begin() or ending end() iterator.
Returns
Current depth of the iterator.
Complexity
Linear in depth that is, at worst linear in size of the other container.
