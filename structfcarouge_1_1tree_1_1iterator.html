Detailed Description

template<class Type, class AllocatorType = std::allocator<Type>>
struct fcarouge::tree< Type, AllocatorType >::iterator

Type to identify and traverse the elements of the container.

The iteration order of the standard iterator is unspecified, except that each element is visited only once.

constexpr size_type fcarouge::tree< Type, AllocatorType >::iterator::depth ( ) const
inlineconstexprnoexcept

Length of the path from the root to the iterated node.

Returns the number of nodes from the root to the currently iterated node, i.e. the current depth of the tree hierarchy. The root has depth of 0, its children have depth 1, etc... The behavior is undefined if *this is the end iterator.

Returns
Current depth of the iterator.
Complexity
Linear in depth.
