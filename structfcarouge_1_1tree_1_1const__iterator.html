Detailed Description

template<typename Type, typename Allocator>
struct fcarouge::tree< Type, Allocator >::const_iterator

Type to identify and traverse the constant elements of the container.

The iteration order of the standard constant iterator is unspecified, except that each element is visited only once.

inlineconstexprnoexcept

Implicit iterator to constant iterator conversion.

The regular, i.e. non-constant, iterator type is implicitely convertible to the constant iterator type per requirements. The conversion is performed through a (non-explicit) converting constructor. A conversion operator is not implemented to avoid disabling implicit move. Consequently the constant iterator is no longer an aggregate type hence disabling aggregate initialization.

Note
No conversion assignment operator is supported following the majority of Standard Template Library (STL) vendors implementation.
