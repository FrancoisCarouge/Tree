Detailed Description

template<TreeMemberIterator Iterator>
struct fcarouge::depth_first_pre_order_iterator_type< Iterator >

A pre-order depth-first tree traversal iterator.

Type to identify and traverse the elements of the container. The iteration order is specified as a pre-order depth-first traversal. Each element is visited only once. The traversal is topologically sorted.

  1. Current node data.
  2. Traverse the left-most child node and data.
  3. Traverse the next child node and data.
